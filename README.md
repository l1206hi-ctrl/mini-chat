# Mini Chat

웹 기반 AI 채팅 앱입니다.  
사용자가 만든 캐릭터와 대화할 수 있도록 만드는 것을 목표로 하고 있습니다.

## Motivation

수능을 마치고 프로그래밍을 처음 접한 후, 1주일 동안 학습하며
제가 좋아하는 앱에서 영감을 받아 이 프로젝트를 만들었습니다.
특히 한국어 대사를 자연스럽게 생성하고, 캐릭터별 톤을 다르게 유지하는 방법을 고민했습니다.

## 실행 방법

1. 이 프로젝트를 다운로드하거나 클론합니다.
2. 의존성 설치:

   npm install

3. 서버 실행:

   node server.mjs

4. 브라우저에서 접속:

   http://localhost:3000
